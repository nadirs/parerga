This blog

I wrote this blog in Python using Flask, a micro-framework, and its siblings Jinja, a templating engine, and Werkzeug, a WSGI utility library.

Text formatting takes advantage of markdown, while code highlighting was made possible by yet another Pocoo product, Pygments.

Comments are powered by Disqus.

The main purpose of this blog was to practice Python and the mentioned Flask framework working on a reasonable project, but also to write down some thoughts and notes about code and software development in general.

Who I am

I am a mostly self-taught developer, interested in high- and low-level programming (including embedded systems' issues, like memory management and signal processing), software life cycle and development patterns.
